AdminPad is a simple note taker for WP site administrator only.

== Description ==

AdminPad allows WP site admin (only) to write down simple note from WP dashboard using simple widget. Deleting this plugin will also delete data saved by this plugin on your database.

Features:
* Light weight plugin.
* Simple user interface.

== Installation ==

1. Upload the "adminpad" folder on your site's "/wp-content/plugins" folder.
2. Activate it from your site's plugin section. You're done!
